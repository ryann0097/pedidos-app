# Pedidos App

Sistema web para gestão de pedidos, inicialmente de uma salgaderia, desenvolvido com Spring Boot e Thymeleaf. Pode ser expandido para outros tipos de gestão de pedidos.

## Funcionalidades atuais
- CRUD de pedidos
- Criação de itens de pedido
- Controle de status do pedido
- Autenticação básica de usuários

## Em desenvolvimento
- Gestão de administradores
- Controle de permissões
- Tratamento global de exceções

## Tecnologias
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Spring Security
- Thymeleaf
- Maven
